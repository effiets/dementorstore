# DementorStore
Rest api e-commerce App  for Harry Potter fans


## Features

### A store for users to :
* browse the store
* search by House (Slyhterin, RavenClaw ..etc) 
* add to Cart
* edit od delete cart
* checkout

### Authentication:

* Login
* Sign Up
* Update info
* Change Password
* Forget Password


### Admins can :
* Add / update / delete product 
* Create a user
* Change Order status

## Technologies
* NodeJS
* Express
* Mongoose

### Dependencies
bcrypt, cors, jsonwebtoken, nodemailer

### DataBase
MongoDB
